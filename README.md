# Frontend Boilerplate
  A boilerplate for modern frontend development

## This boilerplate contains:
 - [x] Lost
 - [x] Stylus
 - [x] Handlebars
 - [x] PostCSS
 - [x] Html5 Boilerplate
 - [x] Gulp Babel

## Instructions

Clone this repository

```sh
git clone https://github.com/adamsalves/frontend-boilerplate.git
```

Install dependencies

```sh
npm install
```

Start application

```sh
npm start
```
